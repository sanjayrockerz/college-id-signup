‚úÖ **Identity Privacy**:

- All user IDs ‚Üí SHA-256(id + ANONYMIZATION_SALT) ‚Üí irreversible 16-char hex tokens
- Conversation IDs ‚Üí same irreversible tokenization
- No raw identifiers ever leave production boundary

‚úÖ **Content Privacy**:

- Message content ‚Üí **NEVER exported** ‚Üí only content length (int)
- Conversation names ‚Üí **NEVER exported** ‚Üí only size distribution
- Attachment files ‚Üí **NEVER exported** ‚Üí only type and size histograms

‚úÖ **Metadata Privacy**:

- Email addresses ‚Üí **NEVER exported** ‚Üí only username length distribution
- Phone numbers ‚Üí **NEVER exported**
- IP addresses, sessions ‚Üí **NEVER exported**

### What Gets Exported (Aggregate Only)

‚úÖ **Safe to export**:

- Histogram buckets: `{50_chars: 12000, 100_chars: 23000, ...}`
- Statistical percentiles: `{p50: 14, p95: 523}`
